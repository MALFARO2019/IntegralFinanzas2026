# Manual de referencia (resumen) – Cash Organizer (Inesoft)
Fecha: 2026-02-25

> Este documento es una **guía resumida** basada en recursos oficiales y un **índice de enlaces**. No reproduce el manual completo.

## 1) Plataformas y concepto
El tutorial oficial indica que Cash Organizer puede usarse como:
- **Web (browser)**
- **Desktop (Windows/Mac/Linux)**
- **Android app**  
y que sincroniza cambios entre plataformas y permite trabajo offline con sincronización posterior.  
Fuente: Getting Started (tutorial). https://www.cashorganizer.com/eng/tutorial_1.html

## 2) Seguridad y sincronización
El tutorial oficial lista como “key features”:
- **Strong encryption** durante almacenamiento y transmisión
- **Automatic synchronization** entre web/Android/desktop
- Posibilidad de **grant access** a otros usuarios en cuentas seleccionadas
- Cloud technologies  
Fuente: Getting Started (tutorial). https://www.cashorganizer.com/eng/tutorial_1.html

## 3) Primeros pasos (alta y base de datos)
Según el tutorial:
1. Entrar al sitio y usar modo Online.
2. Crear cuenta (email + password).
3. Elegir tipo de base de datos.
4. Completar setup inicial.  
Fuente: Getting Started. https://www.cashorganizer.com/eng/tutorial_1.html
