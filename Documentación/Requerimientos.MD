Funcionalidad Comercial:
La idea es que sea una herramienta gratuita para llevar finanzas personales pero que algunos servicios sean a pago.

También espero que existan apis, para que los bancos puedan consultar sobre la cartera de clientes para ofrecerles productos financieros de manera anónima, como refinanciamiento, esos servicios se cobraran

Que exista una opción de metas, como compra de Casa, que la empresas puedan ofrecer igual de forma anónima oferta de todo tipo, esto sería genial para empresas de venta comercial.

Si el usuario acepta una oferta comercial o financiera pueda ponerse en contacto con el proveedor igual de manera anónima, hasta que el usuario lo desee.

Los ingresos de plataforma viene de las empresas que tienen estos servicios.

Se debe tener una plataforma que ofrezca productos en una web o con apis., y se usarán segmentaciones de los usuarios para ofrecer, además de las aceptación de estos de recibir ofertas.

Integración:

Como en Latinoamérica muchos banco no ofrecen conexión a cuentas, se requiere que el usuario pueda re direccionar el correo y la aplicación pueda leer el contenido y realizar la evaluación datos y luego realizar los registros  (gratis).  Debe existir una forma en la administración del sistema de mapear correos de diferentes empresas financieras o bancos,  que cuando llegue determine si se trata de una TRANSACCION conocida y las deje en espera para aprobación, si no logra detectar que es una transacción conocida pueda mapear, para recibirla luego

Algunos banco permiten generar reportes, igual que se permita mapear

En todos los casos evitar duplciación de registros.
